# Site Institucional - ParkMais 🚗💻

Este repositório contém o projeto prático da disciplina de Desenvolvimento para Internet, cujo objetivo é criar um site institucional completo com múltiplas páginas e links de navegação.

## 🎯 Sobre o Projeto
O projeto simula o site da **ParkMais**, uma empresa fictícia focada em administração inteligente de estacionamentos, unindo operações logísticas com soluções de dados.

A atividade atende a dois exercícios principais aplicados de forma integrada:
1. **Estruturação HTML (Exercício 1):** Criação de 5 páginas interligadas (`index`, `empresa`, `servicos`, `equipe` e `contato`) utilizando a hierarquia correta de textos, listas, imagens e âncoras de navegação.
2. **Melhoria Visual (Exercício 2):** Aplicação de **CSS Inline** (atributos `style` diretos nas tags) para aprimorar o aspecto visual do site, incluindo alterações de cor, tamanho de fonte, centralização de conteúdo, bordas arredondadas e cores de fundo.

## 🗂️ Estrutura das Páginas
* `index.html`: Página de introdução.
* `empresa.html`: Propósito e valores institucionais.
* `servicos.html`: Soluções de gestão e automação oferecidas.
* `equipe.html`: Apresentação dos membros e cargos.
* `contato.html`: Informações de atendimento.
